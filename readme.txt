This plugin allows you to implement security captcha form into web forms to prevent spam.

== Description ==

Captcha Bank allows you to protect your website from spam using random characters and numbers which can be used for login, registration, reseting password, comments forms.

== Installation ==

1. Upload `captcha-bank` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
